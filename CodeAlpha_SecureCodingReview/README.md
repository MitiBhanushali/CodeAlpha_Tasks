# Secure Coding Review - CodeAlpha

## Project Overview
This project is part of the CodeAlpha Cyber Security Internship.
A simple Python login program was reviewed to identify security issues
and improve it using basic secure coding practices.

## Objective
- Identify insecure coding practices
- Understand common vulnerabilities in authentication systems
- Apply secure coding techniques to improve the program

## Improvements Made
- Removed hardcoded credentials
- Applied basic input validation
- Improved code readability and structure

## Learning Outcome
- Understanding of secure coding principles
- Hands-on experience in code review
- Awareness of common programming vulnerabilities
