# Phishing Awareness Training - CodeAlpha

## Project Overview
This task is part of the CodeAlpha Cyber Security Internship.
A presentation was created to spread awareness about phishing attacks
and common social engineering techniques.

## Objective
The main objective of this task is to:
- Help users identify phishing emails and fake websites
- Explain how attackers trick users into sharing sensitive information
- Provide best practices to stay safe online

## Content Included
- Types of phishing attacks
- Real-world phishing examples
- Tips to identify suspicious emails and links
- Prevention and safety measures

## Learning Outcome
- Understanding of phishing and social engineering
- Awareness of real-world cyber threats
- Knowledge of safe online practices
